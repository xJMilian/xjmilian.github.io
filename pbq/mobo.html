// Exam Notes

①CPU Socket (LGA): Land Grid Array — Intel's socket design where the pins are on the motherboard, not the CPU. AMD uses PGA (pins on CPU) or LGA (AM5+). The socket type determines CPU compatibility. Obj 3.4
②DIMM Slots: Dual Inline Memory Module slots. ATX boards typically have 2–4 slots. Always consult the QVL (Qualified Vendor List) and install in paired slots (A1+B1) for dual-channel. Obj 3.2
③PCIe x16 Slot: Primary slot for discrete GPUs. Runs 16 lanes at PCIe 4.0/5.0 speeds. Distinguished from x1 slots by its longer length and latch mechanism on the end. Obj 3.4
④M.2 Slot: Supports both SATA and NVMe SSDs in M.2 2280 form factor. Key B supports SATA; Key M supports NVMe PCIe. Check the board spec — not all M.2 slots support both protocols. Obj 3.3
⑤SATA Ports: Serial ATA 3.0 (6 Gb/s). L-shaped connector with no screws required. Commonly used for HDDs, SSDs, and optical drives. The number of ports is controlled by the PCH chipset. Obj 3.3
⑥24-pin ATX Connector: The main power connector from the PSU to the motherboard. Originally 20-pin (ATX 1.x); extended to 24-pin in ATX 2.x. Without this, the board will not POST. Obj 3.5
⑦CMOS Battery (CR2032): A 3V lithium coin cell that maintains BIOS/UEFI settings and the real-time clock (RTC) when AC power is removed. A dead CMOS battery causes the system date to reset and may trigger boot errors. Obj 3.4
⑧Front Panel Header: A 9-pin (10-pin with 1 key) block that connects the case's power button, reset button, HDD LED, and power LED to the motherboard. Miswiring is a common troubleshooting scenario. Obj 3.4
